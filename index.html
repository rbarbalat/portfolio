Gulp

Gulp is a yelp clone that allows users to add businesses to the site, search and review the businesses of other users, respond to reviews and choose favorites. It is a full stack application built with a Flask/PostgreSQL backend and a React/Redux frontend. Checkout the repo.

Cypher

Cypher is a slack clone that allows users to create and join teams and channels. Users can communicate with each through within channels with by posting public messages and with privately with each other through direct messages. Cypher is a full stack application built with a Flask/PostgreSQL backend and React/Redux frontend by a team of four. It utilizes Socket.IO to implement direct and channel messaging. Checkout the repo.

FourLegsGood

FourLegsGood is a MeetUp clone that allows users to create groups, events, venues for groups and assign venues to particular events. It is a full stack application built with a Flask/PostgreSQL backend and a React/Redux frontend. Checkout the repo.
